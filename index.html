(I write code)

Groupr is a web application that is designed to help shy students find groups in large college classes. It presents a tinder-like interface, allowing users to choose which indivuals they'd like to work with, and then creates the most compatable groups. It was created for the 2019 CruzHacks hackathon and also presented at the Santa Cruz New Tech Meetup the following month.

lk is a text-based game that I made after watching Bandersnatch. It's frustratingly difficult and exists mostly to mess with the player. It's still in a pretty early stage.

Lk is still in progress, and is intentionally written as weirdly as possible. It runs by holding a JSON object full of code, and hot-swapping functions from that object into its own codebase.

The Second Harvest FoodFinder was a project that I worked on in order to assist the local Second Harvest Food Bank nonprofit. The website allows members of the community to see when which of the 50+ food distribution sites are open at any given time, with a map. It pulls from a google calendar that can be easily mantained by members of the nonprofit.
